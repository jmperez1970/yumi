Herramienta YUMI

Introducción

YUMI (Your Universal Multiboot Installer) es una herramienta que permite crear unidades USB booteables con múltiples sistemas operativos. Su funcionalidad principal es facilitar la instalación y prueba de diferentes distribuciones de Linux, herramientas de diagnóstico, sistemas de rescate e incluso versiones de Windows desde una sola unidad USB. A diferencia de otros programas similares, YUMI permite agregar varias imágenes ISO en el mismo pendrive sin necesidad de formatearlo cada vez. Esto lo convierte en una herramienta ideal para técnicos y administradores de sistemas, ya que facilita la instalación y gestión de múltiples sistemas operativos desde un solo dispositivo.

Este documento detalla el proceso paso a paso para crear un pendrive booteable utilizando YUMI, asegurando que puedas disponer de una herramienta versátil para instalar y reparar sistemas operativos.

Materiales necesarios

Para completar este procedimiento, necesitarás:

  • Un pendrive de al menos 8GB de capacidad (recomendado 16GB o más para varias ISOs).

  • YUMI (descargable desde pendrivelinux.com).

  • Imágenes ISO de los sistemas operativos o herramientas que desees instalar.

    • Descargar ISO Windows 11
    • Descargar ISO Ubuntu 24.04.1 LTS

  • Un ordenador con Windows para ejecutar YUMI.

Pasos detallados

A continuación se detalla paso a paso el proceso para bootear con YUMI un Pendrive:

Descargar YUMI

Accede a la página oficial de YUMI y descarga la versión más reciente.

Ejecutar YUMI

No requiere instalación. Simplemente abre el archivo descargado.

Hacemos doble click en el archivo descargado y se carga el programa. Pincha en I Agree para aceptar las condiciones de la licencia:

Seleccionar la unidad USB

Conecta el pendrive y elige la letra de unidad correspondiente en YUMI.

Preparar el dispositivo (solo la primera vez)

Si es la primera vez que deseamos bootear con YUMI el pendrive, debemos preparar el pendrive. Esto solo se hace la primera vez y la preparación del pendrive implica eliminar todo la información que hay dentro.

A continuación YUMI te informa de que se va a eliminar toda la información del pendrive:

Escoger el sistema operativo

En el menú desplegable, selecciona la ISO de Windows o la ISO de la distribución de Linux, herramienta de rescate u otro sistema operativo que desees instalar.

Vamos a empezar metiendo la ISO de Windows 11 que tenemos que tener ya descargada:

Cargar el archivo ISO

Pulsa el botón "Browse" y selecciona la imagen ISO previamente descargada.

Iniciar la creación del USB booteable

Haz clic en "Create" y espera a que el proceso finalice. Este proceso puede tardar entre 5 y 15 minutos, dependiendo del tamaño de la imagen ISO y la velocidad del pendrive.

Añadir más ISOs (opcional)

Puedes repetir el proceso para agregar múltiples sistemas en el mismo USB.

Probar el USB

Reinicia el equipo, accede al menú de arranque y verifica que el pendrive sea reconocido.

Errores comunes y soluciones

Al utilizar YUMI para crear unidades USB booteables, pueden surgir algunos problemas comunes que dificulten el proceso. A continuación, se presentan algunos de los errores más frecuentes junto con sus soluciones para facilitar su resolución.

  • YUMI no detecta el pendrive: Asegúrate de que está correctamente formateado en FAT32.
  • El equipo no arranca desde el USB: Revisa la configuración del BIOS/UEFI y habilita el arranque desde USB.
  • Error al cargar la imagen ISO: Verifica que el archivo ISO esté correctamente descargado y que sea compatible con YUMI.

Conclusión

Con YUMI, puedes crear fácilmente un USB booteable con varios sistemas operativos y herramientas de mantenimiento, como Ubuntu, Windows 10, Linux Mint, Hiren's BootCD y herramientas de diagnóstico como MemTest86. Esto te permite instalar, diagnosticar y reparar equipos de manera eficiente, ahorrando tiempo y evitando la necesidad de grabar múltiples discos o USBs. Es una herramienta indispensable para cualquier técnico en sistemas informáticos.
